# Organizador de arquivos com Python

## Descrição
Projeto de automação em Python que organiza os arquivos automaticamente de acordo com as suas extensões.

## Funcionalidades
- Seleção de pasta através de interface gráfica
- Identificação automática da extensão dos arquivos
- Criação automática de pastas por categoria
- Movimentação automática dos arquivos
- Padronização das extensões (maiúsculas/minúsculas)

## Tecnologias Utilizadas
- Python 3
- Biblioteca "os" (manipulação de arquivos e diretórios)
- "tkinter" (interface para seleção de pastas)

## Estrutura de Organização

### Os arquivos são organizados nas seguintes categorias:

- Imagens: ".jpg", ".jpeg", ".png", ".gif", ".bmp"
- Planilhas: ".xlsx", ".xlsm", ".csv"
- PDFs: ".pdf"
- Texto: ".txt", ".docx"

## Para executar

1. Baixe ou clone o repositório.
2. Execute o arquivo Python:
   organizador_python.py.
3. Uma janela será aberta para selecionar a pasta que deseja organizar.
4. Após selecionar a pasta, os arquivos serão organizados automaticamente por suas extensões.

## Aprendizados

- Manipulação de arquivos com Python
- Automação de tarefas repetitivas
- Criação e análise de diretórios
- Uso de dicionários para a organização
